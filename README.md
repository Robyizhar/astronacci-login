Project Specifications :
1. Laravel 8 / php 7.4
2. tailwind 3.4.9
3. mysql

Project Setup in Loclhost:
- Clone repository
- run `composer install`
- copy .env.example to .env
- setup database according in your local setting
- run `php artisan migrate
- run `php artisan db:seed
- run npm install
- run npm run dev
